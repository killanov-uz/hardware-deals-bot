# 🚀 Hardware Deals Bot

A Java-based bot that scans Amazon for **premium hardware deals** and sends real-time alerts via Telegram.

---

## 🔥 Features

- 🔎 Scrapes Amazon across multiple categories
- 🧠 Smart filtering (GPU, CPU, SSD, RAM)
- 📊 Price history tracking
- 🚨 Super Deal detection (real discounts)
- 🥇 Ranking system (best deals first)
- 🚫 Anti-duplicate system
- 🤖 Telegram integration (image + link + price)
- 🔁 Fully automated (runs every 10 minutes)

---

## 📦 Categories Covered

- 🖥 GPUs (RTX 30/40, RX 6000/7000)
- 🧠 CPUs (Ryzen / Intel Core)
- 💾 SSD NVMe (1TB+)
- 🧬 RAM (DDR4/DDR5 16GB+)
